Щоб запустити програму:
  1) відкрийте папку MyAssembler.UI -> bin -> Debug
  2) запустіть MyAssembler.UI.exe

У вікні текстового редактора можна писати або власний код, або зчитати код з файлу одним із способів:
  1) File -> Open...-> папка з файлом -> .asm файл
  2) Ctrl+O -> папка з файлом -> .asm файл

Скомпілювати код можна одним із способів
  1) Assembler -> Build
  2) Ctrl+Shift+B

У разі успішної компіляції 
  1) відкриється вкладка Build Protocol, у якій зібрана інформація про результат компіляції. 
  2) заповняться вкладки Labels, Byte Memory Cells, Word Memory Cells з інформацією про наявні в програмі мітки, зарезервовані байти і слова.
  3) у папці Build output у корені проекту з'являться відповідні .bat i .com файли.

У разі помилки компіляції на елементі StatusBar (внизу вікна) буде виведено повідомлення з описом помилки.

Cкомпілювати і (у випадку успішної компіляції) виконати код у середовищі DOSBox можна одним із способів
  1) Assembler -> Run
  2) Ctrl+R

Щоб очистити текстовий редактор виберіть File -> New або натисніть Ctrl+N

Щоб очистити результати попередньої компіляції, виберіть Assembler -> Clear output або натисніть Ctrl+D.

Щоб дізнатися інформацію про програму, виберіть Help -> About program...

ПРИКЛАДИ

1. Hello - виведення на екран повідомлення "Hello, assembler world!".
2. Print2Lines - виведення на екран 2 рядків: +< і BA
3. Math - обчислення арифметичного виразу і виведення на екран результату
